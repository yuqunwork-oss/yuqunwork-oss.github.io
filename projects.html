AI 影像辨識應用系統

  • 問題：將影像辨識模型應用於實務場景，提供即時或離線辨識結果。
  • 做法：使用 Python 與影像處理技術，整合深度學習模型進行推論。
  • 我的角色：模型整合、影像前處理流程設計、系統實作。
  • 成果：完成可實際執行的辨識流程，並產出可視化結果。
  • 技術：Python、OpenCV、YOLO

Web 系統整合專案

  • 問題：建立可實際使用的網站系統，整合前端與資料處理流程。
  • 做法：設計前端頁面，串接後端與 API，完成資料顯示與操作。
  • 我的角色：前端頁面設計、系統流程規劃與整合。
  • 成果：完成可部署於 GitHub Pages 的實務網站。
  • 技術：HTML、CSS、JavaScript、GitHub Pages

小畫家（Canvas 繪圖應用）

  • 做法：使用 HTML5 Canvas 搭配 JavaScript，處理滑鼠事件與繪圖邏輯。
